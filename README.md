# 🍔 Lanchonete Mario: Sabor e Performance

Seja bem-vindo ao repositório da **Lanchonete Mario**! Este projeto é o resultado dos meus estudos práticos em desenvolvimento Front-end, onde transformei o conceito de uma hamburgueria artesanal em uma interface web funcional e atraente.

> "Mais do que apenas código, este projeto representa meu foco em criar experiências digitais que conectam empresas a clientes."

---

## 🎯 O Desafio
O objetivo principal foi construir uma **Landing Page completa** que não apenas apresentasse o menu, mas que também passasse confiança ao cliente. Foquei em criar uma navegação intuitiva e um design que despertasse o apetite, utilizando uma paleta de cores estratégica.

## 🚀 O que eu aprendi construindo este projeto:

* **Estrutura Semântica:** Uso de HTML5 puro para garantir que o site seja acessível e bem interpretado pelos motores de busca (SEO).
* **Agilidade com Bootstrap 5:** Implementação de componentes complexos como o *Carousel* para depoimentos e *Cards* de preços, reduzindo o tempo de desenvolvimento sem perder a qualidade.
* **Design Responsivo:** O desafio de fazer com que o hambúrguer da seção "Hero" e a tabela de preços ficassem perfeitos tanto em um monitor 4k quanto na tela de um celular.
* **Integração de Bibliotecas:** Gerenciamento de ícones com Font Awesome e fontes personalizadas via Google Fonts para elevar o nível visual da aplicação.

## 🛠️ Tecnologias e Ferramentas
* **Linguagens:** HTML5 e CSS3.
* **Framework:** Bootstrap 5.1 (Grid System, Navbar, Cards e Carousel).
* **Ícones:** Font Awesome (Ícones de marcas e utilitários).
* **Tipografia:** Google Fonts (Alfa Slab One para títulos e Montserrat para leitura).
